# miniature-modelling

## About <a name = "about"></a>

Hobbyist website showcasing custom-made models, deployed to https://miniature-modelling.netlify.app/

- Next.js, React, SASS
- Contentful

## Installing <a name = "installing"></a>

Development

```
npm install
npm run dev
default port 3000
```

Production

```
npm install
npm run deploy
```
